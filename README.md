# kkoutsilis.github.io

Personal blog built with GitHub Pages and the minima theme.

## Adding a new post

1. Create a file in `_posts/`.
2. Name it `YYYY-MM-DD-slug.md`.
3. Add front matter like this:

```md
---
layout: post
title: "Post title"
date: 2026-05-28 17:30:00 +0100
categories: [posts]
---
```

1. Write the post below the front matter.

Example: `_posts/2026-05-28-my-first-post.md`
